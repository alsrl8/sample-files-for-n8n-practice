Claude Code 설치

Windows CMD(명령 프롬프트) 기준

Step 1. 명령 프롬프트(CMD) 열기
시작 메뉴 → "cmd" 검색 → 클릭
Step 2. Git 설치
winget install --id Git.Git -e --source winget
Step 3. Claude Code 설치
CMD를 닫고 새로 열어주세요 (Git 경로 반영을 위해)
curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd
Step 4. 설치 확인
CMD를 다시 닫고 새로 열어주세요
claude --version
* PowerShell에서 "스크립트를 실행할 수 없습니다" 오류가 나면 CMD(명령 프롬프트)를 사용하세요.
* 각 단계 사이에 CMD를 닫고 새로 여는 이유: 새로 설치된 프로그램의 경로가 반영되려면 터미널 재시작이 필요합니다.
