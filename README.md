# n8n 워크플로우 연습용 샘플 데이터

한국콜마 오픈랩 14팀의 n8n 워크플로우 실습에 사용되는 샘플 데이터와 쇼케이스 자료입니다.

## 구성

| 디렉토리 | 내용 | 설명 |
|----------|------|------|
| `/` (루트) | Mock data (xlsx/txt) | 14팀 실습용 목 데이터. SAP/SRM/MES 등 실제 시스템 대신 엑셀로 시뮬레이션 |
| `showcase/` | 쇼케이스 페이지 | 14팀 워크플로우 소개 — 노드 흐름, 요구사항 정의서, 시연 영상, JSON 다운로드 |
| `showcase/json/` | Workflow JSON | n8n에서 import 가능한 워크플로우 정의 파일 14종 |
| `showcase/gifs/` | 시연 영상 | 워크플로우 실행 녹화 (추가 예정) |

## 쇼케이스 페이지

https://alsrl8.github.io/sample-files-for-n8n-practice/showcase/

14팀의 워크플로우를 한눈에 탐색할 수 있는 정적 페이지입니다.

- 팀별 워크플로우 노드 흐름
- 요구사항 정의서 (과제 배경, 구현 방향, 연동 시스템)
- 시연 영상 (GIF 모달)
- 워크플로우 JSON 다운로드 (n8n import용)

## 워크플로우 JSON 사용법

1. `showcase/json/` 에서 원하는 워크플로우 JSON을 다운로드
2. n8n 에디터에서 **Import from File** 선택
3. 다운로드한 JSON 파일 선택
4. Credential(Slack, xAI Grok 등)과 파일 경로를 본인 환경에 맞게 수정

## 데이터 출처

- **Mock data**: 각 팀의 기획서를 기반으로 생성한 가상 데이터 (실제 운영 데이터 아님)
- **Workflow JSON**: n8n에서 14팀 과제를 구현한 예제 워크플로우를 API로 export
- **시연 영상**: 각 워크플로우를 실행하고 Playwright + ffmpeg로 녹화
